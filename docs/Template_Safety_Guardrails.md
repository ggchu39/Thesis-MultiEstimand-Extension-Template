# Template safety guardrails

This repository is a **public template**.

## What is included
- Toy/simulated data generators
- Reusable R helpers
- Quarto workflows (Track A and Track B)
- Generic documentation and reporting skeletons

## What is NOT included
- Any real datasets, participant-level exports, or derived real outputs
- Any unpublished results, effect sizes, or dataset-specific variable names
- Any manuscript text intended for submission

## Do not commit
- `outputs/` folders or rendered reports produced from real data
- Data files (`.csv`, `.tsv`, `.rds`, `.xlsx`, etc.) unless they are explicitly toy/simulated examples intended for publication
- Anything that could enable re-identification or reconstruction of private datasets
